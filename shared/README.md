# Shared

Общий слой для ботов в этом репозитории.

Сюда стоит выносить только код, который действительно общий для нескольких
проектов: контракты событий для админ-бота, небольшие Telegram-утилиты,
общие форматтеры и безопасные адаптеры к платежам.

Не стоит складывать сюда runtime-файлы, `.env`, базы данных или большие части
конкретного бота. Каждый бот остаётся отдельным сервисом со своим окружением.
