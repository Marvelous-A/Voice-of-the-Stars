# TG Bots Monorepo

Репозиторий теперь хранит связанные Telegram-проекты в одном дереве:

```text
Voice-of-the-Stars/
  main.py                  # Voice of the Stars, пока в корне для совместимости деплоя
  AdminBot/                # общий админ-бот
  EchoBot/                 # ЭХО
  NeboBot/                 # Небо рядом
  shared/                  # общий код и контракты между ботами
```

Текущий серверный деплой Voice ожидает основной бот в корне репозитория, поэтому
`main.py` и связанные файлы Voice пока не перенесены в подпапку. Это осознанный
переходный шаг: админ-бот, ЭХО и Небо уже лежат отдельными каталогами, а Voice
останется совместимым с существующим `/root/Voice-of-the-Stars` и `/home/bot`.

Правило для дальнейшей разработки:

- каждый бот живёт в своей папке и запускается отдельным `systemd`-сервисом;
- общение с `AdminBot` делается через маленькие адаптеры и общие контракты, а не
  импортом всего бота целиком;
- runtime-данные (`.env`, базы, очереди, кэши, сессии) не коммитятся;
- общие модули добавляются в `shared/` только когда они нужны минимум двум ботам.

Серверные рабочие директории остаются раздельными:

```text
/home/bot/                # Voice runtime
/home/admin-bot/          # AdminBot runtime
/var/lib/echo-dialog-bot/ # Echo runtime data
/var/lib/nebo-alert-bot/  # Nebo runtime data
```
